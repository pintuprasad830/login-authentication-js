# 🔐 Login Authentication Project

A simple login system built using JavaScript that validates username and password.

## 🚀 Features
- User login form
- Username & password validation
- Shows success message on correct login
- Shows alert for invalid credentials

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
